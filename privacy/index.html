Политика конфиденциальности

Настоящая политика конфиденциальности определяет порядок обработки персональных данных пользователей сайта https://ekatvideo.ru.

Оператором персональных данных является самозанятый гражданин, оказывающий услуги видеосъёмки и видеомонтажа.

1. Какие данные обрабатываются
Сайт не осуществляет сбор персональных данных пользователей через формы.
Персональные данные могут быть получены только при добровольном обращении пользователя по указанному на сайте номеру телефона.
К таким данным относится номер телефона и иная информация, сообщаемая пользователем в ходе общения.

2. Цели обработки данных
Персональные данные используются исключительно для связи с пользователем, обсуждения условий сотрудничества и оказания услуг.

3. Условия обработки
Персональные данные не передаются третьим лицам, не используются для рассылок и не распространяются.
Обработка осуществляется только в объёме, необходимом для взаимодействия с клиентом.

4. Отсутствие автоматического сбора данных
Сайт не использует формы сбора данных, системы аналитики (включая cookies) и иные инструменты отслеживания пользователей.

5. Права пользователя
Пользователь вправе запросить уточнение или удаление своих персональных данных, а также отказаться от дальнейшего взаимодействия.

6. Контактная информация
По всем вопросам, связанным с обработкой персональных данных, пользователь может обратиться по номеру телефона, указанному на сайте.

Настоящая политика действует бессрочно и может быть изменена без предварительного уведомления.
